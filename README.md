# django-CRUD

<!-- setup > -->
- go to django-CRUD folder
- cd django-crud
 <!--create virtual env  -->
 - python3 -m venv crudenv
 - source crudenv/bin/activate (Ubuntu 24.4)
 - pip install django
 <!-- check env working or not -->
 - pip freeze

 <!-- how to run  -->
 - cd crud
 - python3 manage.py runserver
